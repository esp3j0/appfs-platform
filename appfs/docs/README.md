# Documentation Index

This directory keeps two kinds of project documents:

1. **Milestone docs** in `v1/`, `v2/`, `v3/`, and `v4/`
2. **Working plans** in `plans/`

Historical milestone files keep their original names for traceability. The recommended reading order is defined by the index files below rather than by raw filenames.

## Naming Rules

To reduce confusion:

1. `v1` / `v2` / `v3` / `v4` are **milestone archives**
2. `plans/` contains **working design docs**
3. Raw historical filenames are preserved, but new top-level references should prefer these index files
